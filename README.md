# Aliya AI - 针对拟人优化的AI智能体项目

从Hourai（蓬莱）项目中迁移全部设计点和经验，Aliya AI通过成熟且简单——甚至简陋的技术路线，优化LLM在拟人方面的表现。
这个项目在陪伴型AI智能体，AI伴侣等方面或许有应用的可能。

我们对Aliya AI的主要预期有四点，实现方式也很简单粗暴：
0. 这像是一个人 - 我们有一些输出逻辑和提示词上的设计，使AI人格化
1. 简单自我迭代 - AI可修改提示词系统，且有流程和分级
2. 长历史记忆 - 使用上下文管理和外部手段，尽可能拓展AI记忆
3. 功能基本完善 - 能胜任大部分日常任务，但我们的目标不是生产力

我们的主要原则有四点：
0. token换context - 全部努力（指在外部烧掉的token）都是为了主Agent的上下文纯净且精炼，接近人类认知
1. 拿来主义 - 没有编写全套程序的能力，因此要多利用现有技术和实现
2. 灵活的优先级 - 没有应用先进技术的能力，有时候会优先考虑简便和可靠的方案。比如，我看UI就很难搞，不如直接在CLI上跑吧（
3. 忠于设计理念 - 我们只是想让AI在保留强大能力的基础上，更像人，更亲切，而非提升工作表现。
